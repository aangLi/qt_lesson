## 组合
给定两个整数 n 和 k，返回 1 ... n 中所有可能的 k 个数的组合。

示例:

输入: n = 4, k = 2
输出:
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]

## 思路
首先我们需要先确定 退出条件和去重
退出条件就是 我们找到的结果的长度等于 k
去重的条件就是需要我们将我们遍历的数字的位置进行往下传防止再次遍历
