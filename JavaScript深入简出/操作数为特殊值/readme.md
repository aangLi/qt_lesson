# NaN
    NaN 值和任何值都不相等。
- !
    NaN 表示 false
- &&
    存在 NaN 返回 NaN
- ||
    两个 NaN 返回 NaN
- 加减乘除
    如果有一个是 NaN 都返回 NaN

# null 与 undefined
    在比较相等时，是一样的，但是全等时，是不一样的，通常情况下表达的数字为0，在比较相等或者全等的时候，是不会先默认转为任何形式的值。

# Infinity
    表示为很大的数字，基本上的加减乘除都回返回本身，除了对本身和0进行操作有时候会返回 NaN 值

# label 语句
    label 语句一般和循环体相结合使用，伴随 break 与 continue 的使用，相当于给循环体进行一个标记，用于多重循环。
